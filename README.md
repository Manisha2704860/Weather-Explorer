# Weather-Explorer:
-   Weather Explorer is a web-based weather application built using HTML, CSS, and JavaScript. It allows users to search for any city and view real-time weather information along with a short-term forecast. The application fetches live data from the OpenWeatherMap API and displays it in a clean, user-friendly interface.
# Features:

-  Search weather by city name

-  Displays current temperature in Celsius

-  Shows weather description and condition icons

-  Readable text-based weather summary box

-  4-day weather forecast

-  Dynamic background changes based on weather conditions

-  Error handling for invalid city names or network issues

-  Responsive and modern UI design

# Technologies Used:

-  HTML for structure

-  CSS for styling and layout

-  JavaScript for API integration and interactivity

-  OpenWeatherMap API for live weather data
